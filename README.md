# منصة الاجتماعيات - الأستاذ أحمد بوعمود

## التحقق
هذه نسخة ثابتة Static Site تعمل على GitHub/Vercel بدون قاعدة بيانات.

## طريقة النشر الصحيحة
ارفع محتوى المجلد نفسه إلى مستودع GitHub، وليس المجلد الخارجي فقط.

## الروابط
- الصفحة الرئيسية: index.html
- لوحة الإضافة: admin.html أو /admin/

## طريقة إضافة مورد
1. افتح لوحة الإضافة.
2. املأ الخانات.
3. انسخ الكود الناتج.
4. افتح assets/data/resources.json في GitHub.
5. أضف الكود داخل القائمة قبل القوس الأخير ].
6. ضع الصورة في assets/img بنفس الاسم.

ملاحظة: هذه اللوحة تولد الكود فقط، ولا تنشر تلقائياً دون خدمة خارجية أو ربط GitHub API.
